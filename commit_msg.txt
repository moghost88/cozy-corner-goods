fix: remove buggy live email check, keep Firebase verification flow

Removed all disify.com API code and the disposable domain blocklist
from Auth.tsx. They caused false positives (blocking real Gmail/Hotmail
addresses) and were unreliable.

Email validation is now:
1. Zod format check on submit (instant, reliable)
2. Firebase sendEmailVerification after signup (proves the inbox is real)
   - User gets a link in their inbox
   - Must click it before accessing the app
   - ProtectedRoute + EmailVerificationWall enforce this
